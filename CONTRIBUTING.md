# Contributing

Contributions are welcome! Here are some guidelines:

## Adding a new library

- Make sure the library is an HTTP **client** library for Python.
- Add it to the table in `README.md` following the existing format.
- Include the shields.io badges for project health.
- Verify the correct GitHub `owner/repo` path and CI workflow filename.

## Guidelines

- Search existing entries before adding a new one to avoid duplicates.
- One library per pull request.
- Keep descriptions factual and concise.
